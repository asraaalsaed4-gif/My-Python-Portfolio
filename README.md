## Python Data Cleaning Project (Pandas Library)

This repository demonstrates a core data auditing and cleaning skill using Python's Pandas library.

### Project Goal:
To efficiently read raw data from a CSV file, identify inconsistencies (duplicates and missing values), and output a clean, refined dataset.

### Technical Skills Demonstrated:
* **File I/O:** Reading data from CSV files (`pd.read_csv`).
* **Data Auditing:** Identifying and dropping duplicate rows (`.drop_duplicates()`).
* **Data Preprocessing:** Handling missing data (if any) (`.dropna()`).
* **Output:** Saving the cleaned results to a new CSV file (`.to_csv`).

### File Status:
* `data_final.csv`: The initial raw dataset (contains 1 duplicate row).
* `clean_final.csv`: The final cleaned dataset (duplicate row successfully removed).
